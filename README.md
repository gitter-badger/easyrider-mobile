# EasyRider

Aplikacja śledząca trasy rowerzystów. Wersja pre-alpha ;)

## Przygotowanie do developmentu przy pomocy przeglądarki

```bash
sudo npm install -g --upgrade cordova ionic
make setup
cordova platform add browser
ionic run browser
```
